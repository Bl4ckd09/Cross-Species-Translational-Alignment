# Data setup — files not tracked in this repo

This repo ships **code + the linkage cohort + small derived tables + the results table**.
The large expression matrices are **intentionally not committed** — for two reasons:

1. **Size** — the raw GEO download and processed matrices are hundreds of MB (GitHub is the wrong home).
2. **Redistribution terms** — third-party expression (especially Open TG-GATEs / LSDB) should be
   fetched from source, not re-hosted. See [README → Provenance](README.md#provenance--sources).

Everything below regenerates from the scripts plus **one public download**.

---

## What's excluded (and how to get it)

| Path | Size | What it is | How to obtain |
|---|---|---|---|
| `data/_raw/GSE57815_series_matrix.txt.gz` | ~263 MB | DrugMatrix rat-liver expression, raw GEO series matrix (normalised `VALUE`) | **download** (see below) — the only external file the pipeline needs |
| `data/_raw/GSE57815_family.soft.gz` | ~470 MB | GEO family SOFT (extra metadata) | optional — **not used** by the current scripts |
| `data/expression/drugmatrix_liver_expr.parquet` | ~381 MB | probe × sample matrix, subset to cohort + controls | generated by `scripts/retrieve_expression.py` |
| `data/signatures/drugmatrix_liver_logfc.parquet` | ~43 MB | per-compound logFC signatures (treated − vehicle) | generated by `scripts/build_signatures.py` |
| `data/features/structure_ecfp4_2048.parquet` | ~1 MB | ECFP4 (Morgan r=2, 2048-bit) structure fingerprints | generated by `scripts/structure_embed.py` (auto-run) |
| *(future)* TG-GATEs `E-MTAB-799 / 800` | — | 2nd expression source (not yet wired in) | ArrayExpress/BioStudies (see below) |

Already **in** the repo, so no download needed: `tox21.csv` (MoleculeNet Tox21 labels),
`master_cohort.csv`, `data/*_keyed.csv`, `data/chemicals.Rds` (DrugMatrix compound table).

## Quick start — reproduce end-to-end

```bash
# 0. environment (Python 3.11; no GPU / PyTorch needed for the ECFP4 default)
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt

# 1. download the one external file: DrugMatrix rat-liver expression (GEO GSE57815, ~263 MB)
mkdir -p data/_raw
curl -L -o data/_raw/GSE57815_series_matrix.txt.gz \
  https://ftp.ncbi.nlm.nih.gov/geo/series/GSE57nnn/GSE57815/matrix/GSE57815_series_matrix.txt.gz

# 2. build the artifacts (each writes into data/, all git-ignored)
python scripts/retrieve_expression.py   # -> data/expression/drugmatrix_liver_expr.parquet (+ manifest, coverage)
python scripts/build_signatures.py       # -> data/signatures/drugmatrix_liver_logfc.parquet (+ labels, report)
python scripts/run_experiment.py         # -> data/results/results_table.csv (+ run_config.json)
                                         #    also builds data/features/structure_ecfp4_2048.parquet
```

`retrieve_expression.py` **reads the pre-downloaded** `GSE57815_series_matrix.txt.gz` (step 1) — it does
not fetch it for you. `run_experiment.py` triggers `structure_embed.featurize()` and `expr_embed`
internally, so you do not run those two directly.

Expected coverage on the current single-source (DrugMatrix liver) pass: **177 / 613** labelled
compounds, 31,099 probes → the `results_table.csv` you see committed here.

## External datasets

### DrugMatrix — **required** (single-source v1)
- **GEO GSE57815** — DrugMatrix Affymetrix rat liver (platform GPL1355), normalised intensities.
- Series matrix (the file the pipeline needs, ~263 MB):
  `https://ftp.ncbi.nlm.nih.gov/geo/series/GSE57nnn/GSE57815/matrix/GSE57815_series_matrix.txt.gz`
- Family SOFT (optional, ~470 MB):
  `https://ftp.ncbi.nlm.nih.gov/geo/series/GSE57nnn/GSE57815/soft/GSE57815_family.soft.gz`
- Landing page: https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE57815
- License: NIEHS / NTP (US public domain).

### TG-GATEs — for the **next step** (2nd source, not yet integrated)
- **E-MTAB-799** (rat in-vivo liver) and **E-MTAB-800** — Open TG-GATEs on ArrayExpress / BioStudies.
  - https://www.ebi.ac.uk/biostudies/arrayexpress/studies/E-MTAB-799
  - https://www.ebi.ac.uk/biostudies/arrayexpress/studies/E-MTAB-800
- Both are Affymetrix Rat 230 2.0 → same probe space as GSE57815 → ready for cross-source ComBat.
- Entry point: `scripts/retrieve_expression.py::fetch_tggates` (stubbed as task #5).
- License: Open TG-GATEs, **LSDB Archive** — **do not re-host**; fetch from source.

## Where the big artifacts belong (if you want to publish them)

- **Processed features** (logFC, PCA, ECFP4) → a **HuggingFace Dataset** (`upload_folder`) or Git LFS.
- **Trained model** (fusion head; ChemBERT upgrade planned) → a **HuggingFace Model** repo (`push_to_hub`) with the AUC/AUPRC model card.
- **Raw third-party matrices** → never re-hosted here; always fetched from GEO / ArrayExpress via the script above.

This keeps the GitHub repo small and reproducible: `python scripts/retrieve_expression.py` regenerates
the data from source rather than shipping a data dump.
